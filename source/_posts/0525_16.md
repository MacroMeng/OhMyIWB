根据 [Entire 官网](https://entire.io) 的介绍：
> **Entire 是全球下一个开发者平台，让代理与人类能够协作、学习并共同交付。 **

它具有三个核心特点：
- Agent 版本控制
  一种针对 Agent Coding 时代下的 SVC 方式
- 语义推理层
  我没看懂（？），但是它讲述的集中于 AI 上下文管理这方面。
- AI 原生软件开发生命周期
  一种船新的开发方式。

Entire 由 GitHub 前任 CEO Thomas Dohmke 创立，由于 GitHub 之后诸多原因，许多员工投奔 Entire。不过 Entire 目前看来并没有和。GitHub 有直接的竞争关系，如果你想知道，可以去它的官网一探究竟。

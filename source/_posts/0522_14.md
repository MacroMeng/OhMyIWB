**Cloudflare 仪表盘和 Cloudflare API 服务问题**

- 更新 - 我们已经回滚了受影响的服务，现在看到受影响的 API 和仪表盘开始恢复。
  我们预计随着与该服务关联的内存缓存（不是我们的CDN缓存）重新填充，恢复将是渐进的。
  一旦错误率恢复到正常水平，我们将再次提供最新进展。
  五月 22, 2026 - 16:22 世界协调时间
- 监测 - 我们正在努力安全地回滚支撑我们API权限系统的受影响服务。我们预计随着与该服务关联的内存缓存（不是我们的CDN缓存）重新填充，恢复将是渐进的。
  五月 22, 2026 - 16:19 世界协调时间
- 已识别 - 问题已被识别，正在修复中。
  五月 22, 2026 - 15:41 世界协调时间
- 调查 - Cloudflare 正在调查 Cloudflare 仪表盘及相关 API 的问题。 

> 这些问题不会影响通过我们的CDN、安全功能或已部署的计算与存储服务来提供流量。

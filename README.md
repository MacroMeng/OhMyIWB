OhMyIWB 欢迎任何人参与讨论。您可以先在智教联盟论坛中分享想法、得到反馈，也可以在那里向项目启动者 @MacroMeng (STCN@MacrosMeng) 提请申请。若您在社区中声望十足（？），可以直接获得 Collaborator 权限。即使您是新朋友，我们也欢迎看到您的表现——按照标准流程提出 Pull Request 即可。
